    # 3.1、HTML介绍

    • HTML : 是一种用于创建网页的标记语言，可以使用HTML创建网页，用浏览器打开会自动解析；
    • HTML 是由标签和内容构成；
    • HTML 代码结构；
    <html>
<head>
<title>文档的标题</title>
</head>
<body>
文档的内容...
</body>
</html>

    # 3.2、文本格式化标签

    标签 描述
    <br/> 换行
    <h1> ~ </h1> 标题，定义标题字体大小，1最大，6最小
    <p>...</p> 段落
    <i>...</i> 斜体
    <cite> <cite> 引用
    <b>...</b> 加粗
    <strong>...</strong> 强调加粗
    <del></del> 删除线
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>pyenv.cc</title>
</head>
<body>
<!-- <br> 换行符 -->
    print("hello world")</br>
    fmt.Println("hello world")</br>
    console.log("hello world")</br>
<!-- <p> 段落 -->
    <p>golang</p>
    <p>python</p>
    <p>javascript</p>
<!-- <i> 斜体 -->
    <i> Django </i>
    <i> Gin </i>
    <br>
<!-- <del> 删除线 -->
    <del>被逼无奈学JS</del>
</body>
</html>

    # 3.3、列表标签

    标签 描述 参数
    <ul> 无序列表 type=disc 默认实心圆 square 实心方块 circle 空心圆
    <ol> 有序列表 type=1 默认数字，其他值 : A/a/I/i/1
    <li> 列表项目 在有序列表和无序列表中用
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>pyenv.cc</title>
</head>
<body>
    <h1>后端开发技术栈</h1>
    <!--  无序列表  -->
    <ul type="square">
        <li>Django</li>
        <li>Flask</li>
        <li>Gin</li>
    </ul>
    <!--  有序列表  -->
    <ol>
        <li>Django</li>
        <li>Flask</li>
        <li>Gin</li>
    </ol>
</body>
</html>

    # 3.4、超链接标签

    • 超链接标签 : <a href="网址">
    属性 描述
    href 指定链接跳转地址
    target 链接打开方式，常用值 : _blank 打开新窗口
    title 文字提示属性
    name 定义锚点
    <!DOCTYPE html>
<html lang="en" xmlns="http://www.w3.org/1999/html">
<head>
    <meta charset="UTF-8">
    <title>pyenv.cc</title>
</head>
<body>
    <h1>后端开发技术栈</h1>
    <a href="http://pyenv.cc">点击跳转,站内跳转</a></br>
    <a href="http://pyenv.cc" target="_blank">点击跳转,打开新窗口</a>
</body>
</html>

    # 3.6、表格标签

            <td>www.aliangedu.cn</td>

            <td>CentOS7</td>
    </tr>
    </tbody>
</table>

    # 3.7、表单标签

    • 表单标签 ：
    属性 描述
    action 提交的目标地址URL
    method 提交方式 : get默认和post
    enctype 编码类型:
    application/x-www-form-urlencoded 默认值，编码字符
    multipart/form-data 传输数据为二进制类型，如提交文件
    text/plain 纯文本的传输
    • 表单项标签 :
    属性 描述
    type text : 单行文本框
    password : 密码输入框
    checkbox : 多选框
    radio : 单选框
    file : 文件上传选择框
    button : 普通按钮
    submit : 提交按钮
    reset : 重置按钮
    name 表单项名，用于存储内容值
    value 表单项的默认值
    disabled 禁用该元素
    checked 默认被选中，值也是checked
    <!DOCTYPE html>
<html lang="en" xmlns="http://www.w3.org/1999/html">
<head>
    <meta charset="UTF-8">
    <title>pyenv.cc</title>
</head>
<body>
    <h1>表单标签</h1>
    <form action="#">
        用户名 : <input type="text"> <br>
        密  码 : <input type="password"> <br>
        所在地 : <br>
            <input type="radio" name="city" value="beijing">北京<br>
            <input type="radio" name="city" value="shanghai">上海<br>
        爱好 : <br>
            <input type="checkbox" name="hobby" value="study">学习<br>
            <input type="checkbox" name="hobby" value="meimei">美女<br>
        提  交 : <input type="submit">
    </form>
</body>
</html>

    # 3.8、列表标签

    • 下拉列表标签 :
    属性 描述
    name 下拉列表的名称,用于存储下拉值
    disabled 禁用该元素
    multiple 设置可以选择多个项目
    size 指定下拉列表中的可见行数
    • 下拉列表选项标签 :
    属性 描述
    value 选项值
    selected 默认下拉项
    <!DOCTYPE html>
<html lang="en" xmlns="http://www.w3.org/1999/html">
<head>
    <meta charset="UTF-8">
    <title>pyenv.cc</title>
</head>
<body>
    <h1>表单标签</h1>
    <form action="#">
        用户名 : <input type="text"> <br>
        密  码 : <input type="password"> <br>
        所在地 : <br>
            <input type="radio" name="city" value="beijing">北京<br>
            <input type="radio" name="city" value="shanghai">上海<br>
        爱好 : <br>
            <input type="checkbox" name="hobby" value="study">学习<br>
            <input type="checkbox" name="hobby" value="meimei">美女<br>
        <!--   单选     -->
        性别 : <br>
            <select name="sex" id="sexid">
                <option value="man">男</option>
                <option value="girl">女</option>
            </select>
            <br>
        <!--   多选     -->
        功课 : <br>
            <select name="proj", id="projid" multiple>
                <option value="python">Python</option>
                <option value="golang">Golang</option>
                <option value="js">JavaScript</option>
            </select>
            <br>
        提  交 : <input type="submit">
    </form>
</body>
</html>

    # 3.9、按钮标签

    • 按钮标签 :
    • type 可选值
      • button : 普通
      • submit : 提交
      • reset : 重置
    <!DOCTYPE html>
<html lang="en" xmlns="http://www.w3.org/1999/html">
<head>
    <meta charset="UTF-8">
    <title>pyenv.cc</title>
</head>
<body>
    <h1>普通按钮</h1>
    <input type="button" value="普通按钮"><br>
    <button>普通按钮</button>
    <h1>重置示例</h1>
    <form action="#">
        用户名 : <input type="text"> <br>
        密  码 : <input type="password"> <br>
        <!--    form中的reset按钮会将所有表单内容重置    -->
        <input type="reset">
    </form>
    <input type="submit">

</body>
</html>

    # 3.10、<div> 标签

    • <div> 标签用于在 HTML文档中定义一个区块，常用于将标签集中起来，然后用样式对她们进行统一排版；
    <!DOCTYPE html>
<html lang="en" xmlns="http://www.w3.org/1999/html">
<head>
    <meta charset="UTF-8">
    <title>pyenv.cc</title>
</head>
<body>
    <div style="background: crimson">
        <p>段落1</p>
        <p>段落2</p>
        <p>段落3</p>
    </div>

    <div style="background: cornflowerblue">
        <p>段落4</p>
        <p>段落5</p>
        <p>段落6</p>
    </div>
</body>
</html>
