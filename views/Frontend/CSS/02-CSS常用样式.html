    # 一、CSS 样式语言

    # 1.1、CSS介绍

    • CSS : 是一种用于修饰网页的文本样式语言，还可以配合JavaScript脚本语言动态对网页各元素操作；

    # 1.2、使用方法

    1、内联方式（行内样式）
<p style="color:red">在HTML中如何使用css样式</p>

2、内部方式（内嵌样式），在head标签中使用
<style type="text/css">
p {
color:red;
}
</style>

3、外部导入方式（推荐），在head标签中使用
<link href="main.css" type="text/css" rel="stylesheet"/>

    # 1.3、选择器

    • 选择器 : 需要改变样式的HTML元素;
    • 格式 : 选择器 {属性:值; 属性:值;....}
    • 常见选择器 : 标签选择器、类选择器、ID选择器、派生选择器；

    # 1.3.1、选择器 ：元素

    • 元素选择器 : 使用HTML标签作为选择器，为指定标签设置样式；
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>CSS练习</title>
    <style>
    /*  示例1：h1元素设置样式  */
        h1 {
            color: chocolate;
            font-size: 50px;
        }
    /*示例2：多个元素设置样式*/
        h2 {
            color: gold;
        }
    /*示例3：子元素会继承最高级元素所有属性*/
        body {
            color: cyan;
            font-family: Verdana, serif;   /*字体*/
        }
    </style>
</head>
<body>
    <div>
        <h1>H1</h1>
        <h2>H2.1</h2>
        <h2>H2.2</h2>
    </div>
</body>
</html>

    # 1.3.2、选择器 ：ID

    • ID选择器 : 使用 "id" 作为选择器，为指定id设置样式；
    • 使用格式 : #id名 {样式...}
    • 特点 ：
      • 每个id名称只能在HTML文档中出现一次；
      • 在实际开发中，id一般预留 javacript 使用；
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>CSS练习</title>
    <style>
    /* id选择器 ：针对id设置样式 */
    /* 第二步：针对id设置样式 */
    #div1 {
        color: crimson;
    }
    #h1 {  /*当子属性和父标签属性一致时，子标签属性会覆盖父标签属性*/
        color: darkblue;
    }
    </style>
</head>
<body>
    <!--- 第一步：给标签指定id --->
    <div id="div1">
        <h1 id="h1">H1</h1>
        <h2>H2.1</h2>
        <h2>H2.2</h2>
    </div>
</body>
</html>

    # 1.3.3、选择器 ：类

    • 类选择器 ：使用 "类名" 作为选择器，为指定类设置样式；
    • 使用格式 ：类名 {样式...}
    • 特点 ：每个class名称可以在HTML文档中出现多次；
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>CSS练习</title>
    <style>
    /* 类选择器 ：为指定类设置样式(class名称可以出现多次，是可以复用的) */
    /* 第二步 ：针对类设置样式 */
    .div1 {
        color: chartreuse;
    }
    .h1 {
        color: darkblue;
    }
    </style>
</head>
<body>
    <!--第一步 ：给标签指定类-->
    <div class="div1">
        <h1 class="h1">H1</h1>
        <h2>H2.1</h2>
        <h2>H2.2</h2>
    </div>
    <div class="div1">
        <h1 class="h1">H1</h1>
        <h2>H2.1</h2>
        <h2>H2.2</h2>
    </div>
</body>
</html>

    # 1.3.4、选择器 ：派生

    • 派生选择器 ：依据元素在其位置的上下文关系来定义样式；

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>CSS练习</title>
    <style>
    /* 派生类选择器 */
    /* 第二步 ：针对类设置样式 */
    .div1 {
        color: chartreuse;
    }
    .h1 {
        color: darkblue;
    }
    /* 派生用法 */
    .div1 p {
        color: crimson;
    }
    </style>
</head>
<body>
    <!--第一步 ：给标签指定类-->
    <div class="div1">
        <h1 class="h1">H1</h1>
        <h2>H2.1</h2>
        <h2>H2.2</h2>
        <p>say hello</p>
    </div>
</body>
</html>

    # 1.4、常用属性

    # 1.4.1、内边距和外边距

    • padding (内边距) : 钻戒到盒子内边框的距离；
    • margin (外边距) : 钻戒盒子距离桌子边缘的距离；
    • border : 钻戒盒子边框宽度；
    属性 描述
    padding 设置四边的内边距
    padding-top 上内边距
    padding-right 右内边距
    padding-bottom 下内边距
    padding-left 左内边距
    • 示例 ：
      • padding : 10px 5px 15x
    属性 描述
    margin 设置四边的外边距，使用方法同padding
    margin-top 上外边距
    margin-right 右外边距
    margin-bottom 下外边距
    margin-left 左外边距

    • div 设置宽和高的效果
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>CSS练习</title>
    <style>
    /* 派生类选择器 */
    .cc1 {
        background: limegreen;
        /*设置宽高*/
        width: 200px;
        height: 200px;
    }
    .cc2 {
        background: red;
        /*设置宽高*/
        width: 200px;
        height: 200px;
    }
    </style>
</head>
<body>
    <div class="cc1">
        <h1 class="h1">H1</h1>
        <p>p.1</p>
        <p>p.2</p>
        <p>say hello</p>
    </div>

    <div class="cc2">
        <h1 class="h2">H2</h1>
        <p>p.2</p>
        <p>p.2</p>
        <p>say hello</p>
    </div>
</body>
</html>

    # 1.4.2、字体 font-*

    属性 描述 值
    font-size 设置字体的大小 xx-small、x-samil、small、medium、large、x-large、xx-large 从小到大，默认值为 medium；
    length 固定长度，例如12px
    font-family 字体系列，可以写多个,如果第一个不支持，使用下一个 Microsoft YaHei
    font-weight 设置字体的粗细 normal 默认值
    bold 粗体
    bolder 更粗
    lighter 更细
    font-style 字体样式 normal 正常
    italic 斜体
    oblique 倾斜的字体
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>字体</title>
    <style>
        .cc {
            color: red;
            font-size: 17px;
            font-weight: bold;
        }
    </style>
</head>
<body>
    <div class="cc">
        <p>段落1</p>
        <p>段落2</p>
    </div>
</body>
</html>

    # 1.4.3、文本

    属性 描述 值
    color 字体颜色 颜色名称，例如 red
    十六进制值，例如 #ff0000
    rgd 代码，例如 rgd(255,0,0)
    text-align 文本对齐方式 left左边
    right右边
    center中间
    justify两端对齐文本效果
    text-decoration 文本修饰 onoe默认,定义标准的文本，例如去掉超链接下划线
    line-through 删除线
    underlibe 文本下加一条线
    text-overflow 文本溢出后展示效果 clip 修剪文本
    ellipsis 显示省略号来代表被修改的文本
    string 使用给定的字符串来代表被修剪的文本
    letter-spacing 字符间的距离 normal 默认
    length 自定义间距
    line-height 行间的距离(行高) normal 默认
    legnth 设置固定值

    # 1.4.4、边框 border-*

    属性 描述 值
    border 所有边框样式的缩写 示例：border: 1px solid bule; 宽度 样式 颜色
    border-radius 圆角边框 直接写像素
    box-shadow 给元素添加阴影 格式 ：box-shadow: h-shadow v-shadow blur spread color inset ；
    h-shadow : 必选，水平阴影的位置
    v-shadow : 必选，垂直应用的位置
    blur : 可选，模糊程度
    spread : 可选，阴影的大小
    color : 可选，阴影的颜色
    inset : 可选，从外层的阴影(开始时)改变阴影内侧阴影
    示例1 : box-shadow: 1px 2px 3px 1px #c2c2c2；
    示例2: box-shadow: 0 5px 20px 0 #e8e8e8;

    # 1.4.5、背景 background-*

    属性 描述 值
    background-color 背景颜色 颜色名称，例如 red
    十六进制值，例如 #ff0000
    rgd 代码，例如 rgd(255,0,0)
    background-image 背景图片 url('URL') 图片路径
    none 不显示背景图片
    background-repeat 设置是否及如何重复背景图片 repeat : 默认，背景图像将在垂直方向和水平方向重复
    repeat-x : 背景图像将在水平方向重复
    repeat-y : 背景图像将在垂直方向重复
    no-repeat : 背景图像将仅显示一次
    background-position 背景图片的位置 left、top、top right 、center left 、center center、center right 、bottom left 、bottom center、bottom right
    %x %y 水平位置和垂直位置
    background-size 背景图片的尺寸 length : 背景的高度和宽度，例如80px 60px
    percentage : 以父元素的百分比设置背景图像的高度和宽度，例如 50% 50%
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>IMAGE</title>
    <style>
        .tt {
            width: 300px;
            height: 300px;
        }
        .t1{
            width: 300px;
            height: 300px;
            background-image: url("./image/hnkz1.png");
            background-size: 100px 100px;
            background-repeat: no-repeat;
        }
    </style>
</head>
<body>
    <div class="tt">
        <div class="t1">
        </div>
    </div>
</body>
</html>

    # 1.4.6、Flex 弹性布局

    • 在之前要控制 HTML元素的布局，会用到 padding、margin、postion、float等方法，经过反反复复调试才能实现效果；
    • 自从 Flex 弹性布局出现，一切几乎豁然开朗；
    • 启用Flex 布局只需要在外部元素设置 display : flex 属性；

    • Flex布局有一个隐式的坐标空间，水平方向有一条主轴，垂直方向有一条交叉轴：

    • 改变主轴(横向) 的布局 :

      • justify-content
        • flex-end : 右对齐
        • center : 居中对齐
        • space-evenly : 平分空间
        • space-between : 两端对齐

    • 改变交叉轴(竖向) 的布局 :

      • align-items :
        • flex-end : 靠下对齐
        • center : 居中对齐

    • 调整空间占比(子元素宽度)

      • 例如第一个元素三分之一，第二个元素占三分之二，第三个元素占三分之一 : flex:1 ，flex2, flex1
    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .flex {
            border: 5px solid grey;
            display: flex;
            justify-content: space-around;
        }
        .flex1 {
            width: 300px;
            height: 300px;
            background: green;
        }
        .flex2 {
            width: 300px;
            height: 300px;
            background: magenta;
        }
        .flex3 {
            width: 300px;
            height: 300px;
            background: maroon;
        }
    </style>
</head>
<body>
    <div class="flex">
        <div class="flex1">DIV1</div>
        <div class="flex2">DIV2</div>
        <div class="flex3">DIV3</div>
    </div>
</body>
</html>
